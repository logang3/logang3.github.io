This site is hosted on GitHub Pages. Domain name managed through Hover.

I write every post in MultiMarkDown using the iOS app Drafts 5. I custom-built a few Drafts actions to build each page in HTML and send *.html and *.text versions to Working Copy, then push the changes to GitHub.1
